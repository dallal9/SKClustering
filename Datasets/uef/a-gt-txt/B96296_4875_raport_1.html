Random List Generator¶

A function to generate N random numbers between minN and maxN. The default value for minN is zero, and for maxN is maximum. Python random library by default uses a random seed so it generates different output for each function call and it uses a uniform selection. The function returns a list of N random intger numbers.

In [3]:
def get_randN(N, minN=0, maxN=maxnum):
    Nnums = []
    for i in range(N):
        Nnums.append(random.randint(minN, maxN))
    return Nnums

Sorting Algorithm¶

I implemented a bubble sort function, which takes a list of integers as input and outputs a new sorted list. Bubble sort algorithm works by simply swap any wrong ordered adjacent elements. It is a very simple sorting algorithm, but has a worst complexity and average complexity of n^2.

The following function is a simple implementation for bubble sort algorithm. It takes a list of integers as input and returns a new list with the integer numbers sorted.

In [4]:
def bubble_sort(n_list):
    sorted_list = list(n_list)
    n = len(n_list)
    for i in range(n):
        for j in range(0, n-i-1):
            if sorted_list[j] > sorted_list[j+1]:
                sorted_list[j], sorted_list[j +1] = sorted_list[j+1], sorted_list[j]
    return sorted_list

Experiments Generator¶

The following function is constructed to make it easy to run different experiments. In the function we can set the N, minN, maxN for the random range. The function allows the repetition of certain combination by using flag “rep”. In [5]:
def sort_b(N, minN=0, maxN=maxnum, rep=1, unique=True):
    times = []
    Nnums = get_randN(N, minN, maxN)
    for i in range(rep):
        if unique:
            Nnums = get_randN(N, minN, maxN)
        t1=time.time()
        sorted_list = bubble_sort(Nnums)
        t2=time.time()
        times.append(float(t2-t1))
    return times,np.mean(times),np.std(times)

EX5¶

$f(n) = 8n^3 + n$

$g(n) = n^3$

Using Asymptotic Analysis

$$\lim_{n \to \infty} \frac {f(n)}{ g(n)} = \lim_{x \to \infty} \frac {8n^3 + n}{ n^3}= \lim_{x \to \infty} \frac {8n^2 + 1}{ n^2} = \frac{8}{1}=8 $$
$$0< \frac {f(n)}{ g(n)}<\infty$$

since the limit is constant then this proves that $$f(n)= \theta(g(n))$$

which menas $$ c_1g(n) \leq f(n) \leq c_2g(n) $$ for all $$ n>n_0 $$

$$ c_1 n^3 \leq 8n^3+n \leq c_2 n^3 $$

dividing by n^3 $$ c_1 \leq 8+\frac{1}{n} \leq c_2 $$

then for $$n_0 \geq 1$$ $$c_1=8$$ and $$c_2 =9$$

def shuffle_list(n_list):
    shuffled_list = list(n_list)
    n = len(n_list)
    for i in range(n):
        j  = random.randint(0,(n-1-i))
        shuffled_list[j], shuffled_list[n-1-i] = shuffled_list[n-1-i], shuffled_list[j]
    return shuffled_list
  • then created a function that checks if a list is sorted by checking if every element is less than the next element if not then it is not sorted

def is_sorted(n_list):
    for i in range(len(n_list)-1):
        if n_list[i]>n_list[i+1]:
            return False
    return True

def random_sort(n_list):
    while True:
        new_list=shuffle_list(n_list)
        if is_sorted(new_list):
            return new_list
